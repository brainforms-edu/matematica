# BrainForms
<div align="justify">
BrainForms é um sistema educacional para auxiliar estudantes do ensino médio no aprendizado de fórmulas matemáticas e físicas, áreas que frequentemente apresentam dificuldades devido ao receio de alunos com a área de exatas e a métodos de ensino desatualizados. O projeto conta com uma área de conteúdos organizados por nível escolar de forma dinâmica e com base em materiais acadêmicos, bem como uma ferramenta que resolve fórmulas automaticamente e apresenta o passo a passo detalhado das soluções, permitindo que o aluno entenda a teoria e a prática no ensino. Além disso, apresenta questionários organizados por nível de dificuldade com um vasto banco de questões e um espaço de mini games educativos para estimulação da prática e raciocínio rápido, assim os usuários avaliam seu desempenho e consolidam seus conhecimentos. Por fim é destacado um sistema de conquistas para incentivar o aluno a evoluir na plataforma e um painel de dashboard para monitoramento de suas atividades. Dessa forma, o BrainForms possui o objetivo de apoiar os estudos, reduzir dificuldades e tornar o aprendizado de exatas mais descontraído e eficiente para os estudantes.
</div>

## Sobre

Este repositório reúne duas frentes principais:

- **Conteúdo didático**: explicações organizadas por tópicos, exemplos e referências BNCC.
- **Fórmulas interativas**: página com seleção de fórmulas e cálculo guiado para estudo.

O BrainForms tem como objetivo auxiliar estudantes por meio de:
- 📚 Conteúdos organizados e didáticos
- 🧮 Cálculo automático de fórmulas
- ❓ Questionários interativos (em breve)
- 🎯 Aprendizagem prática e progressiva

## Como Executar Localmente

### Opção 1: Abrir direto no navegador

1. Abra o arquivo `conteudo-geometria-plana/index.html` no navegador.
2. Para a parte de fórmulas, abra `formula-geometria-plana/geometria-plana.html`.

### Opção 2: Servidor local (recomendado)

Use uma extensão como **Five Server** no VS Code ou outro servidor estático.

Exemplo com VS Code:

1. Clique com o botão direito em `conteudo-geometria-plana/index.html`.
2. Selecione **Open with Five Server**.

## Funcionalidades

- Navegação por tópicos com sidebar.
- Destaque automático do tópico ativo durante rolagem.
- Layout responsivo para mobile, tablets, notebooks e monitores.
- Sessões didáticas com imagens e tabelas.
- Bloco de referência BNCC com modal.
- Página de fórmulas com cálculos e resolução passo a passo.

## Convenções de Organização

- CSS separado por página.
- JavaScript separado por responsabilidade (conteúdo, fórmulas, ajustes mobile, botões de informação).
- Assets de imagem por módulo (`Imagens/` em cada pasta principal).
